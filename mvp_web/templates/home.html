Turn your bank & wallet exports into a monthly summary

This tool ingests spreadsheets and PDFs, normalizes transactions, applies your category rules, and builds simple budget metrics. Everything runs in a temporary browser session—nothing permanent unless you download the outputs.

Supported today: Mercado Pago account exports and Santander Argentina (checking account movements and Visa card statements), as XLSX or PDF. Other banks or formats are not wired in yet—results may be wrong or empty.

  1. Download your latest bank / wallet exports (XLSX or PDF).

  2. Upload them in step 1. Files send as soon as you confirm in the file dialog.

  3. Set budgets in step 2 — monthly targets per category (and an overall cap). Add or remove categories here.

  4. Run the pipeline in step 3. Use Advanced → Category rules to add keywords so transactions map to each category (required for new categories you create).

Regex (optional) in Advanced category rules: besides plain keywords, you can use regex patterns for messy bank descriptions. See step 3.

3 · Run pipeline

Imports all movements from your uploads (no month filter). The output filename uses the month of the latest transaction date.

Category rules = keywords
For each category, you list words that may appear in the transaction description (bank text). The pipeline assigns the first category whose keyword is found (case-insensitive). Open the JSON editor to add or change keywords and optional regex patterns.
Only changed budget numbers in step 2? You can run the pipeline without opening Advanced. Open Advanced when you need to fix how transactions are labeled.
